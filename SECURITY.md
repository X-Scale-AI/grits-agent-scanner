# Security Policy

## Reporting a Vulnerability

Do not open a public GitHub issue for security vulnerabilities.

Email: security@xscaleai.com

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix

We will respond within 48 hours and aim to release a fix within 7 days for
critical issues.

## Scope

This policy covers:
- `grits-agent-scanner` -- scanner script
- `grits-agent-secure` -- fixer script
- All files in this repository

## Supported Versions

| Version | Supported |
|---|---|
| 0.3.x | Yes |
| < 0.3 | No |
